Este repositorio contiene el código fuente para el sitio web principal de **RIBN.digital**, una agencia de desarrollo web premium basada en Buenos Aires. 

## 🛠 Stack Tecnológico

- **Framework:** Next.js con App Router
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion
- **Lenguaje:** TypeScript
- **Despliegue:** Vercel

## 🗺 Secciones del Sitio

La estructura principal del sitio incluye las siguientes rutas:
- `/` - Home
- `/trabajos` - Portfolio de proyectos destacados
- `/metodo` - Nuestro proceso de trabajo
- `/contacto` - Formulario de contacto y comunicación

## 🎨 Estilo Visual

- **Concepto:** Diseño *Editorial Tech*
- **Tipografías:** Space Grotesk + Inter
- **Paleta de Colores:** Fondo negro con gradientes elegantes de cyan a purple.

## 💻 Desarrollo Local

Para correr este proyecto en tu entorno local, sigue estos pasos desde la terminal:

```bash
cd ribn-web
npm install
npm run dev
```

Una vez iniciado, abre [http://localhost:3000](http://localhost:3000) para ver la aplicación en vivo.
